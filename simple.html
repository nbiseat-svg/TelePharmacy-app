TelePharmacy Application

Quick Start Guide

To run the TelePharmacy app:

Step 1: Open terminal in project folder
Step 2: Start backend server
cd server && npm start
Step 3: Open new terminal
cd frontend && npm run dev
Step 4: Visit in browser
http://localhost:5173

Need Help?

If you see this page, the full app isn't running yet.

Follow the steps above to start the TelePharmacy application.
